回测{% if selected_label %} - {{ selected_label }}{% endif %}

仅用于回测第一交易日的临时 current_units，默认 5%；不覆盖策略中的长期底仓、补仓初始仓位或加仓倍数。

回测收益计算口径

价格和参数选用

  • 信号、MA150、区间判断使用复权价 Adj Close。
  • 成交、估值、成本、最新价格和收益计算使用原始价 Close。
  • 百分比模式下，qty / units 表示仓位比例或成本权重，不是股数。
  • 历史回测初始成本使用回测窗口第一天原始价；current_avg_cost 只用于实盘监控。
  • 累计投入仓位 = 回测起始持仓 + 回测期间所有买入仓位之和。

收益率计算

  • 分红收益率 = 分红现金贡献 ÷ 累计投入仓位；分红按除息日、按当时持仓比例入账。
  • 交易实现收益率 = 已实现卖出收益贡献 ÷ 累计投入仓位；卖出收益贡献 = 卖出仓位 × (卖出原始价 ÷ 当时持仓成本 - 1)。
  • 持仓收益率 = 期末浮盈贡献 ÷ 累计投入仓位；期末浮盈贡献 = 期末市值权重 - 期末剩余持仓成本权重，不含分红和已实现收益摊薄。
  • 综合收益率 = 分红收益率 + 交易实现收益率 + 持仓收益率 = 总收益贡献 ÷ 累计投入仓位。
  • 摊薄后持仓成本会把分红现金贡献和已实现交易收益贡献摊回期末剩余持仓成本；若收益已覆盖原始成本，页面允许按负成本展示。
  • 期末持仓收益率按程序当前口径计算；摊薄后持仓成本可显示为负数，但负成本只作为展示，不反向影响综合收益率、分红收益率、交易实现收益率和持仓收益率。
